内容简介：
    为校园或社区提供一套高效、透明的设备报修系统，用户可快速提交故障信息，维修人员接单处理，全程可追踪，提升后勤服务效率。
    主要功能：
    用户端：故障上报（文字+图片+定位）、工单状态查看、评价反馈
    维修端：工单接收、状态更新（待处理/处理中/已完成）、打卡签到
    管理端：工单分配、人员调度、数据统计（维修时效、高频故障）
    自动派单与超时提醒机制
    支持多角色权限管理（普通用户、维修员、管理员）
    可选技术：
    前端：Vue3 + TypeScript + Vant（移动端优先）
    后端：Spring Boot + MyBatis-Plus / NestJS
    数据库：MySQL + Redis（缓存工单状态）
